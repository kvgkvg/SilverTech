# Template Images

Place official reviewed appliance panel images here.

The current MVP seed records reference text placeholders so the backend and
tests can run without private photos. Replace these placeholders with real
control-panel images before a real-device demo.

Required image rules:
- Include only the appliance control panel, not people or private scenes.
- Keep one image per template layout.
- Store button coordinates in `button_coordinates.sample.json` or the database
  seed file.
